`valar.inference` is a thin layer over Valar's hosted inference endpoints. A
call POSTs your JSON payload exactly as given and returns the raw JSON response as
a `dict` — nothing is reshaped. The value it adds is correlation: when a
[Trace](/traces-sdk) is active, the wrapper tags the request with the current
span and agent so the model call shows up in the right place on the timeline.

```python theme={null}
import valar

resp = valar.inference.responses.create(
    model="zai-org/GLM-5",
    input="Say hello in one sentence.",
)

chat = valar.inference.chat.completions.create(
    model="zai-org/GLM-5",
    messages=[{"role": "user", "content": "hello"}],
)
```

## How correlation works

The wrappers look for a Trace in two places: an explicit `trace=` argument, or the
current Trace set by `init()`. When one is found, they add the
`X-Valar-Trace-Id` header along with the active span and agent context, and the
dashboard pins the call to that exact point in the run.

```python theme={null}
with trace.agent("reviewer", name="Reviewer", role="reviewer"):
    with trace.span("draft"):
        # Attributed to this agent and span automatically.
        valar.inference.responses.create(model="zai-org/GLM-5", input="...")
```

Pass `trace=` to aim at a specific Trace. With no Trace active at all, the call
still works — it's just plain, uncorrelated inference.

## Methods

Both methods share the same signature. The keyword arguments below are reserved
by the wrapper; everything else you pass becomes the request body.

<ParamField path="**payload" type="dict">
  The request body — `model`, `input`, `messages`, and so on. Sent verbatim.
</ParamField>
<ParamField path="trace" type="Trace" default="None">
  A specific Trace to correlate with. Defaults to the current Trace.
</ParamField>
<ParamField path="headers" type="Mapping[str, str]" default="None">
  Additional request headers, merged into the outgoing request.
</ParamField>
<ParamField path="timeout" type="float" default="None">
  Request timeout in seconds.
</ParamField>

### responses.create

```python theme={null}
def create(
    *,
    trace: Trace | None = None,
    headers: Mapping[str, str] | None = None,
    timeout: float | None = None,
    **payload,
) -> dict
```

POSTs `payload` to `/v1/responses` and returns the raw JSON response dict.

### chat.completions.create

```python theme={null}
def create(
    *,
    trace: Trace | None = None,
    headers: Mapping[str, str] | None = None,
    timeout: float | None = None,
    **payload,
) -> dict
```

POSTs `payload` to `/v1/chat/completions` and returns the raw JSON response dict.

## Errors

A failed call raises `valar.InferenceError` — for example when `stream=True` is
passed or no API key is configured — or `valar.InferenceHTTPError` for a non-2xx
response. The error classes and their attributes live in [Errors](/traces-sdk-errors).

## Bringing your own client

<Warning>
  Streaming is not supported in v0. Passing `stream=True` raises
  `valar.InferenceError`.
</Warning>

To stream, or to use your own HTTP or OpenAI client, call the endpoint directly
and attach the Trace id yourself with
[`valar.trace.headers()`](/traces-sdk#headers):

```python theme={null}
import json, os, urllib.request
import valar

valar.trace.init(name="raw-client")

api_urls = {
    "prod": "https://api.valarhq.ai",
    "dev": "https://dev.valarhq.ai",
    "staging": "https://staging.valarhq.ai",
}
mode = os.environ.get("VALAR_MODE", "prod").strip().lower() or "prod"
api_url = os.environ.get("VALAR_API_URL", "").strip() or api_urls[mode]

headers = valar.trace.headers({"Content-Type": "application/json"})
headers["Authorization"] = "Bearer " + os.environ["VALAR_API_KEY"]

req = urllib.request.Request(
    api_url.rstrip("/") + "/v1/responses",
    data=json.dumps({"model": "zai-org/GLM-5", "input": "hello"}).encode(),
    headers=headers,
    method="POST",
)
```

<Note>
  The v0 Trace SDK is deliberately small. It does not ship native async helpers,
  streaming inference wrappers, an agent framework, orchestration, swarms, memory
  graphs, tool abstractions, or an OpenAI client factory.
</Note>
